# Tic-Tac-Toe ❌⭕

A classic, interactive Tic-Tac-Toe game built for the web. Play locally against a friend directly in your browser with a clean and responsive user interface.

## 🚀 Features
* **Two-Player Mode:** Play alternate turns (X and O) on the same device.
* **Win/Draw Detection:** Automatically detects and announces when a player wins or if the game ends in a tie.
* **Reset Functionality:** Easily clear the board to start a new game with a single click.

## 🛠️ Tech Stack
* **HTML:** Structure of the game board and web page.
* **CSS:** Styling, grid layout, and visual feedback for interactions.
* **Vanilla JavaScript:** Game logic, turn switching, and win condition checking.

## 📁 File Structure
* `index.html` — The main layout and structure of the game board.
* `style.css` — Contains all the visual styling and layout rules.
* `script.js` — Handles the core game logic, user interactions, and state management.

## 💻 How to Run Locally

Since this is a standard frontend application running entirely in the browser, you don't need any complex installation steps or local servers.

1. Clone this repository to your local machine:
```
git clone https://github.com/mythical-invader/tic-tac-toe.git
```
2. Navigate to the project folder.
3. Simply double-click the index.html file to open it in your default web browser and start playing!
